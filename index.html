Rectangle

Definition

A rectangle is a quadrilateral with four right angles and opposite sides of equal length.

Key Properties

  • Opposite sides are equal in length
  • All interior angles are 90°
  • Opposite sides are parallel
  • Two lines of symmetry
  • Diagonals are equal in length
  • Diagonals bisect each other
  • Can be inscribed in a circle if and only if the diagonals are equal

Formulas

Formula Description
A = l × w Area
P = 2(l+w) Perimeter
d = √(l²+w²) Diagonal length
θ = 90° Interior angles

Square

Definition

A square is a regular quadrilateral with four equal sides and four right angles.

Key Properties

  • All sides are equal in length
  • All interior angles are 90°
  • Opposite sides are parallel
  • Four lines of symmetry
  • Diagonals are equal in length
  • Diagonals bisect each other at right angles
  • Inscribed circle touches all four sides
  • Can be inscribed in a circle

Formulas

Formula Description
A = s² Area
P = 4s Perimeter
d = s√2 Diagonal length

Rhombus

Definition

A quadrilateral with all sides of equal length.

Key Properties

  • All sides are equal in length
  • Opposite angles are equal
  • Diagonals bisect each other at right angles
  • Diagonals bisect the angles
  • Two lines of symmetry along the diagonals
  • Area can be found using the diagonals
  • The distance from any point on one diagonal to the nearest side is constant

Formulas

Formula Description
A = ½ × diagonal₁ × diagonal₂ Area
P = 4 × side Perimeter
A = s² × sin(θ) Area using side and angle
h = a × sin(θ) Height
d₁ = 2s × cos(θ/2) First diagonal
d₂ = 2s × sin(θ/2) Second diagonal

Triangle

Definition

A polygon with three sides and three angles.

Key Properties

  • Sum of interior angles is 180°
  • Sum of exterior angles is 360°
  • Types by sides:
    • Equilateral: All sides equal
    • Isosceles: Two sides equal
    • Scalene: No sides equal
  • Types by angles:
    • Right: One 90° angle
    • Acute: All angles < 90°
    • Obtuse: One angle > 90°
  • Has three medians that intersect at the centroid

Formulas

Formula Description
A = ½ × base × height Area
P = ab+bc+ac Perimeter
s = (ab+bc+ac)/2 Semi-perimeter
\( ab^2 + bc^2 = ac^2 \) Pythagoras' theorem

Circle

Definition

A perfectly round shape where every point is equidistant from the center.

Key Properties

  • All radii are equal
  • Has infinite lines of symmetry
  • Has no corners or edges
  • Diameter is twice the radius
  • Most efficient shape in terms of perimeter-to-area ratio
  • Every line through the center creates equal halves
  • All inscribed angles subtending the same arc are equal

Formulas

Formula Description
π = 3.14 or \( \frac{22}{7} \) Pi
A = πr² Area
C = 2πr Circumference
d = 2r Diameter
A = πd²/4 Area using diameter
C = πd Circumference using diameter

Parallelogram

Definition

A quadrilateral where opposite sides are parallel and equal in length.

Key Properties

  • Opposite sides are equal and parallel
  • Opposite angles are equal
  • Adjacent angles are supplementary (sum to 180°)
  • Diagonals bisect each other
  • Any line through the center divides the area in half
  • Can be rotated 180° onto itself
  • The sum of the squares of the diagonals equals the sum of the squares of all four sides

Formulas

Formula Description
A = base × height Area
P = 2(ab + bc) Perimeter
θ₁ + θ₂ = 180° Adjacent angles
d1 = √(a2 + b2 + 2ab cos(θ)) First diagonal
d2 = √(a2 + b2 - 2ab cos(θ)) Second diagonal

Trapezoid

Definition

A quadrilateral with at least one pair of parallel sides.

Key Properties

  • Has one pair of parallel sides (bases)
  • Non-parallel sides are called legs
  • Angles adjacent to each leg are supplementary
  • In an isosceles trapezoid:
    • Non-parallel sides are equal
    • Angles at each base are equal
    • Diagonals are equal
  • The line segment joining the midpoints of the non-parallel sides is parallel to the bases

Formulas

Formula Description
A = ½ × (base₁ + base₂) × height Area
P = base₁ + base₂ + leg₁ + leg₂ Perimeter
m = (b₁ + b₂)/2 Median length
h = √(l² - ((b₁-b₂)/2)²) Height (isosceles)
